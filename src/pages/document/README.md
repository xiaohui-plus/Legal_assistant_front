# 文书模块页面

## 参数填写页 (form.vue)

### 功能概述

参数填写页允许用户根据选定的法律文书模板填写必要的参数，并生成标准化的法律文件。

### 主要功能

#### 1. 动态表单生成 (需求 7.1)
- 根据模板定义动态生成表单字段
- 支持多种字段类型：
  - 文本输入 (text)
  - 多行文本 (textarea)
  - 数字输入 (number)
  - 日期选择 (date)
  - 下拉选择 (select)
  - 单选 (radio)
  - 多选 (checkbox)

#### 2. 实时验证 (需求 7.2)
- 在字段失去焦点时触发验证
- 支持多种验证规则：
  - 必填验证
  - 长度/范围验证
  - 正则表达式验证
  - 自定义验证函数

#### 3. 特殊字段验证
- **金额字段** (需求 7.3): 不能为 0
- **日期字段** (需求 7.4): 不能选择过去的日期

#### 4. 表单状态管理
- **生成按钮启用条件** (需求 7.5):
  - 所有必填字段已填写
  - 所有字段验证通过
  - 无验证错误

#### 5. 草稿功能
- **暂存表单** (需求 7.7): 将当前填写内容保存到本地存储
- **恢复草稿** (需求 7.8): 重新打开模板时自动恢复之前暂存的内容

#### 6. 文书生成 (需求 7.6)
- 验证表单完整性
- 调用生成接口
- 跳转到预览页
- 清除草稿数据

### 使用方式

#### 页面跳转
```javascript
uni.navigateTo({
  url: `/pages/document/form?templateId=${templateId}`
})
```

#### 参数说明
- `templateId`: 模板ID（必填）

### 数据流

```
1. 页面加载 → 获取模板ID
2. 加载模板数据 → 初始化表单
3. 加载草稿数据 → 恢复之前填写的内容
4. 用户填写 → 实时验证
5. 暂存/生成 → 保存数据/跳转预览
```

### 本地存储

#### 草稿数据
- **存储键**: `form_drafts_{templateId}`
- **数据格式**: `Record<string, any>`
- **生命周期**: 
  - 点击"暂存表单"时保存
  - 成功生成文书后清除
  - 用户可手动清除

### 验证规则示例

```typescript
// 文本长度验证
{
  min: 2,
  max: 20,
  message: '姓名长度应在2-20个字符之间'
}

// 数字范围验证
{
  min: 1,
  max: 1000000,
  message: '金额必须在1-1000000之间'
}

// 正则表达式验证
{
  pattern: '^1[3-9]\\d{9}$',
  message: '请输入有效的手机号'
}

// 自定义验证函数
{
  validator: (value) => value !== 0,
  message: '金额不能为0'
}
```

### 错误处理

- 加载模板失败：显示错误提示
- 加载草稿失败：静默失败，使用默认值
- 保存草稿失败：显示错误提示
- 生成文书失败：显示错误提示，保留表单数据
