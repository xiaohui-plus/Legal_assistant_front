# Tab Bar Icons

This directory contains the tab bar icons for the application.

## Required Icons

The following icon files are referenced in pages.json:

- `chat.png` - Chat tab icon (inactive state)
- `chat-active.png` - Chat tab icon (active state)
- `document.png` - Document tab icon (inactive state)
- `document-active.png` - Document tab icon (active state)
- `profile.png` - Profile tab icon (inactive state)
- `profile-active.png` - Profile tab icon (active state)

## Icon Specifications

- **Size**: 81px × 81px (recommended for best display across platforms)
- **Format**: PNG with transparency
- **Color**: 
  - Inactive: #999999 (gray)
  - Active: #165DFF (primary blue)
