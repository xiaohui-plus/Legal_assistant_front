🔧 法律条文查询功能调试

修复了方法名和数据属性名冲突的问题

🐛 问题诊断
发现的问题：
方法名 showLegalArticles() 与数据属性名 showLegalArticles 冲突
解决方案：
将数据属性重命名为 showLegalArticlesPage
🔧 修复内容
// 修复前（有冲突）: data() { return { showLegalArticles: false, // ❌ 与方法名冲突 } } // 修复后： data() { return { showLegalArticlesPage: false, // ✅ 重命名避免冲突 } }
