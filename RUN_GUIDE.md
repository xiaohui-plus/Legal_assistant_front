# 快速运行指南

## ⚠️ 重要提示

当前项目缺少 **uni-cmark** 模块，需要先安装才能正常运行。

## 安装步骤

### 方法一：通过 HBuilderX 安装（推荐）

1. **打开项目**
   - 在 HBuilderX 中打开本项目

2. **安装 uni-cmark 模块**
   - 在项目根目录右键
   - 选择 `uni_modules` -> `从插件市场导入`
   - 搜索 `uni-cmark`
   - 点击安装

3. **等待安装完成**
   - 安装完成后，`uni_modules` 目录下会出现 `uni-cmark` 文件夹

### 方法二：手动下载安装

1. 访问插件市场：https://ext.dcloud.net.cn/plugin?name=uni-cmark
2. 点击"下载插件 ZIP"
3. 解压到项目的 `uni_modules` 目录下
4. 确保目录结构为：`uni_modules/uni-cmark/...`

## 运行项目

安装完 uni-cmark 后，选择运行平台：

### 1. 运行到 H5（推荐用于快速预览）

在 HBuilderX 中：
```
运行 -> 运行到浏览器 -> Chrome
```

或使用快捷键：`Ctrl + R`，然后选择浏览器

**优点：**
- 启动快速
- 调试方便
- 支持热更新

### 2. 运行到微信小程序

**前置条件：**
- 安装微信开发者工具
- 在 manifest.json 中配置小程序 appid

**运行步骤：**
```
运行 -> 运行到小程序模拟器 -> 微信开发者工具
```

### 3. 运行到 App

**前置条件：**
- 连接 Android 手机或启动模拟器
- 开启 USB 调试

**运行步骤：**
```
运行 -> 运行到手机或模拟器 -> 运行到 Android App 基座
```

## 常见问题

### Q1: 提示找不到模块

**错误信息：**
```
Cannot find module '@/uni_modules/uni-cmark'
```

**解决方案：**
按照上面的步骤安装 uni-cmark 模块

### Q2: AI 无法响应

**可能原因：**
1. 未配置 AI 服务的 token
2. 网络连接问题

**解决方案：**
1. 检查 `uni_modules/uni-ai-x/config.uts` 中的配置
2. 查看控制台错误信息
3. 参考 `config.example.md` 配置 AI 服务

### Q3: 页面空白

**检查项：**
1. 确认 uni-cmark 已正确安装
2. 查看控制台是否有报错
3. 尝试重新编译项目

### Q4: 编译错误

**解决方案：**
1. 清理项目缓存：`运行` -> `清理项目缓存`
2. 重新编译
3. 如果还有问题，重启 HBuilderX
