# 法律AI系统快速开始指南

## 🚀 5分钟快速上手

### 第一步：启动后端服务

```bash
# 确保后端服务运行在 http://localhost:8001
cd /path/to/backend
python main.py
```

### 第二步：运行前端项目

在 HBuilderX 中：
1. 打开项目
2. 点击"运行" → "运行到浏览器" → "Chrome"
3. 或运行到手机/模拟器

### 第三步：注册账号

1. 点击左侧菜单的 **"登录/注册"** 按钮
2. 切换到 **"注册"** 标签
3. 输入信息：
   - 用户名：`testuser`（3-20字符）
   - 密码：`123456`（6-50字符）
   - 邮箱：`test@example.com`（可选）
4. 点击 **"注册"** 按钮
5. 注册成功后，切换到 **"登录"** 标签
6. 输入用户名和密码，点击 **"登录"**

### 第四步：体验功能

#### 💬 法律问答
1. 在主页输入框输入问题，例如：
   ```
   劳动合同试用期最长多久？
   ```
2. 点击发送按钮
3. 查看AI回答、法律依据和建议

#### 📄 文书生成
1. 点击左侧菜单的 **"文书生成"** 按钮
2. 选择一个模板，例如：**"房屋租赁合同"**
3. 按照提示逐步回答问题：
   - 出租方姓名
   - 承租方姓名
   - 房屋地址
   - 租金金额
   - ...
4. 完成后点击 **"下一步"**
5. 生成完成后点击 **"下载文档"**

## 🔍 调试技巧

### 1. 查看网络请求
打开浏览器开发者工具（F12）→ Network 标签
- 查看请求URL
- 查看请求头（Authorization）
- 查看响应数据

### 2. 查看控制台日志
打开浏览器开发者工具（F12）→ Console 标签
- 查看API请求日志
- 查看错误信息

### 3. 查看本地存储
打开浏览器开发者工具（F12）→ Application 标签 → Local Storage
- `legal_ai_access_token`: JWT Token
- `legal_ai_user_info`: 用户信息
- `uni-ai-chat`: 对话列表
- `uni-ai-setting`: 设置信息

## ⚠️ 注意事项

1. **后端服务必须先启动**
   - 确保运行在 `http://localhost:8001`
   - 访问 `/docs` 查看API文档

2. **首次使用需要注册**
   - 用户名：3-20字符
   - 密码：6-50字符
   - 邮箱：可选，需符合邮箱格式

3. **登录后才能使用功能**
   - 问答功能需要登录
   - 文书生成需要登录
   - Token 有效期7天

4. **文档下载**
   - 生成的文档为 Word 格式（.docx）
   - 文档有效期7天
   - 下载后可离线查看

## 🆘 常见问题

### Q1: 登录失败，提示"请求失败"
**A**: 检查后端服务是否启动，访问 `http://localhost:8001/health` 测试

### Q2: 问答无响应
**A**: 
1. 确保已登录
2. 检查浏览器控制台是否有错误
3. 查看后端日志

### Q3: 文书生成失败
**A**: 
1. 确保已登录
2. 检查是否完整回答了所有必填问题
3. 查看后端日志

### Q4: 下载文档失败
**A**: 
1. 检查文档是否已过期（7天）
2. 检查网络连接
3. 尝试重新生成文档

### Q5: Token 过期
**A**: 
- Token 有效期7天
- 过期后会自动清除并提示重新登录
- 重新登录即可
