## 注意事项

### 测试要求
- 每个功能模块必须包含单元测试
- 所有正确性属性必须有对应的属性测试
- 属性测试使用 fast-check 库，至少运行 100 次迭代
- 测试标注格式：`Feature: legal-assistant-app, Property {number}: {property_text}`
- 核心业务逻辑测试覆盖率 ≥ 80%

### 开发规范
- 使用 TypeScript 严格模式
- 遵循 Vue 3 Composition API 规范
- 使用 rpx 单位适配不同屏幕
- 所有网络请求必须有错误处理
- 所有用户输入必须进行验证

### 性能要求
- 首屏加载时间 < 3 秒
- 用户操作响应时间 < 100ms
- 长列表使用虚拟滚动
- 图片上传前必须压缩

### 兼容性要求
- 支持微信小程序基础库 2.10.0+
- 支持 iOS 10.0+
- 支持 Android 5.0+
- 支持主流浏览器（Chrome, Safari, Firefox）

### UI设计规范（基于原型）
- 使用 Tailwind CSS 风格的设计系统
- 主色调：indigo-600 (#4F46E5)
- 背景色：slate-50 (#F8FAFC)
- 文本色：slate-900 (#0F172A)
- 边框色：slate-200 (#E2E8F0)
- 圆角规范：按钮 rounded-xl，卡片 rounded-2xl，标签 rounded-full
- 阴影规范：卡片 shadow-sm，按钮 shadow-md，输入框 shadow-xl
- 间距规范：组件间距 space-y-4 或 space-y-6，内边距 p-4 或 p-6
